# Módulo 4 – SQL Analítico

## Objetivo

Utilizar SQL para transformar datos en información útil, respondiendo preguntas de negocio mediante consultas analíticas, métricas e indicadores.

## Contenidos Trabajados

* SELECT.
* Alias mediante AS.
* DISTINCT.
* WHERE.
* Operadores de comparación (=, >, <, >=, <=).
* Operadores lógicos (AND, OR, NOT, IN).
* ORDER BY.
* LIMIT / TOP.
* GROUP BY.
* HAVING.
* COUNT().
* SUM().
* AVG().
* MIN().
* MAX().
* CASE WHEN.
* Introducción a consultas analíticas orientadas al negocio.

## Herramientas Utilizadas

* MySQL

## Actividades Realizadas

* Construcción de consultas para responder preguntas de negocio.
* Segmentación y filtrado de información.
* Generación de rankings comerciales.
* Construcción de métricas e indicadores.
* Análisis de ventas, clientes y productos.
* Proyecto práctico RetailPro.

## Scripts SQL

Ubicados en la carpeta scripts_sql.

## Aprendizajes Técnicos

* Extracción y filtrado de datos.
* Construcción de consultas analíticas.
* Uso de funciones de agregación.
* Generación de KPIs e indicadores.
* Segmentación mediante CASE WHEN.
* Interpretación de resultados desde una perspectiva de negocio.
* Transformación de datos en información para la toma de decisiones.
