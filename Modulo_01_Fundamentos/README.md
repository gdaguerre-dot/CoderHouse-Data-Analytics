# Módulo 1 – Fundamentos de Data Analytics

## Objetivo

Comprender los conceptos fundamentales del análisis de datos, el rol del Data Analyst y el ciclo de vida de los proyectos analíticos.

## Contenidos Trabajados

- Datos, información y conocimiento.
- Business Intelligence.
- Data Analytics.
- Data Science.
- Metodología CRISP-DM.
- KPIs e indicadores.
- Formulación de preguntas de negocio.

## Herramientas Introducidas

- Excel
- SQL
- Python
- Power BI
- GitHub

## Casos Prácticos

- CoffeeInsight
- RetailPro
- Diseño conceptual de proyectos 

## Competencias Desarrolladas

- Comprensión del ciclo analítico.
- Diseño inicial de KPIs.
- Formulación de preguntas de negocio.
- Comprensión del rol del Data Analyst.
- Introducción al ecosistema de datos.
