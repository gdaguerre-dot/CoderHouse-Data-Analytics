# Módulo 6 – ETL y Transformación de Datos con Power BI

## Objetivo

Aprender a limpiar, transformar y preparar datos para el análisis.

## Contenidos Trabajados

* Power Query.
* ETL.
* Calidad de datos.
* Lenguaje M.
* Transformaciones.
* Modelado dimensional.

## Herramientas Utilizadas

* Power BI
* Power Query

## Actividades Realizadas

* Limpieza de datos.
* Tratamiento de nulos.
* Corrección de tipos de datos.
* Automatización de transformaciones.

## Archivos

* PBIX en carpeta pbix.
* Transformaciones en carpeta power_query.

## Aprendizajes Técnicos

* Procesos ETL.
* Lenguaje M.
* Preparación de datos para análisis.
