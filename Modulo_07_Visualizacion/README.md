# Módulo 7 – Visualización de Datos y Storytelling con Power BI

## Objetivo

Diseñar dashboards efectivos que faciliten la comprensión de los datos y la toma de decisiones mediante principios de visualización, experiencia de usuario y narrativa analítica.

## Contenidos Trabajados

- Diseño de dashboards.
- Jerarquía visual.
- UX aplicada a Business Intelligence.
- KPIs y métricas.
- Interactividad.
- Filtros y segmentadores.
- Tooltips personalizados.
- Bookmarks.
- Navegación mediante botones.
- What-If Parameters.
- Data Storytelling.
- Comunicación de insights.

## Herramientas Utilizadas

- Power BI Desktop
- Power BI Service
- DAX
- Copilot para Power BI

## Actividades Realizadas

### Diseño de Dashboards

- Construcción de reportes interactivos.
- Aplicación del patrón de lectura en Z.
- Uso de revelación progresiva.
- Diseño centrado en el usuario.

### Interactividad

- Filtros y segmentadores.
- Drill-down.
- Filtrado cruzado.
- Bookmarks.
- Navegación personalizada.

### Storytelling

- Identificación de insights.
- Construcción de narrativas analíticas.
- Modelo Contexto → Problema → Solución.
- Presentación de recomendaciones accionables.

## Proyectos del Módulo

### Cafeitaly Dashboard

Diseño conceptual y mockup de dashboard para una cadena de cafeterías, aplicando principios de jerarquía visual y experiencia de usuario.

### Data Coffee Storytelling

Análisis de desempeño comercial mediante narrativa analítica orientada a la toma de decisiones.

Hallazgo principal:

- Las ventas disminuyeron un 15% durante los días de lluvia.
- La incorporación de pastelería artesanal incrementó el ticket promedio de $7,00 a $9,50.
